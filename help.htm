Schema Properties

Property

Type

Count

The count column indicates how many times each property is permitted within a single instance of the parent element. Each Count is expressed as either a single vale or range of values. If the Count entry for a property includes the value 0, the property is optional, otherwise the property is required. If the count entry for a property included the value n, the property may be used more than one time.

  • 1: the property shall be used once
  • 1-n: the property shall be used at least once and may be used more than once
  • 0-1: the property is optional and may be used at most once
  • 0-n: the property is optional and may be used more than once
